# Sarsaの概要  
## 前提
* Sarsa（やQ-learning、DQN）は状態価値ではなく、**行動価値Q(s,a)** を用いる。  
  行動価値Q(s,a)とは、状態sのときに行動aを取ったときに未来に得られる報酬の期待値。  

## Sarsaとは  
* TD法のアルゴリズムの一種  
* 1ステップごとにTD誤差を計算して行動価値を更新していく  
* 更新の際の行動選択と実際の行動選択で同じ行動を取る**方策オン型**の手法
